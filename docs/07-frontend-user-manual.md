## 系統概述

禾一考勤系統是一套完整的企業出勤管理解決方案，支援：
- QR Code 掃描打卡（含 GPS 定位驗證）
- 請假、加班、補打卡申請與審批
- 多層級審批流程
- 出勤報表與統計
- 即時通知系統

### 系統網址
- 前台：`https://your-domain.com/` 或 `http://localhost:5173/`
- 需使用現代瀏覽器（Chrome、Safari、Firefox、Edge）

### 角色說明
| 角色 | 權限說明 |
|------|----------|
| 一般員工 | 打卡、申請請假/加班/補打卡、查看個人報表 |
| 部門主管 | 員工權限 + 審批下屬申請、查看部門報表 |
| HR 管理員 | 主管權限 + 員工管理、假別額度設定 |
| 總經理 | 全公司權限、跨部門審批 |

## 登入與帳戶管理

### 登入步驟
1. 開啟系統網址
2. 輸入員工編號（如：EMP001）
3. 輸入密碼
4. 點擊「登入」按鈕

### 忘記密碼
1. 在登入頁面點擊「忘記密碼」
2. 輸入註冊的 Email
3. 系統會發送臨時密碼到您的信箱
4. 使用臨時密碼登入後，建議立即修改密碼

### 修改密碼
1. 登入後點擊右上角「修改密碼」
2. 輸入舊密碼
3. 輸入新密碼（至少 8 個字元）
4. 確認新密碼
5. 點擊「確認修改」

### 登出
點擊右上角「登出」按鈕即可安全登出系統。

## 打卡功能

### 打卡前準備
1. **開啟定位權限**：首次使用需允許瀏覽器存取位置
2. **確認 GPS 狀態**：頁面會顯示「GPS 狀態: ✅」表示定位成功

### 掃描打卡流程
1. 點擊首頁的「📷 掃描打卡」按鈕
2. 將手機相機對準公司的 QR Code
3. 系統自動辨識後會驗證：
   - QR Code 是否為有效公司
   - 您的位置是否在公司 GPS 範圍內
4. 驗證成功後顯示「打卡成功」

### 打卡類型
- **上班打卡**：當天第一次掃描 QR Code
- **下班打卡**：已完成上班打卡後再次掃描

### 打卡狀態說明
| 狀態 | 說明 |
|------|------|
| 正常 | 在規定時間內完成打卡 |
| 遲到 | 超過上班時間（含寬限時間）才打卡 |
| 早退 | 在下班時間前提早打卡下班 |
| 補打卡 | 透過補打卡申請補登記錄 |

### 無法打卡的情況
- **GPS 未開啟**：請開啟手機定位服務
- **位置超出範圍**：您不在公司允許的打卡範圍內
- **無效的 QR Code**：QR Code 不屬於任何已登記公司
- **今日已打卡完成**：上下班打卡都已完成

## 請假申請

### 申請步驟
1. 點擊首頁「📝 申請請假」按鈕
2. 填寫請假表單：
   - **假別**：選擇請假類型（特休、病假、事假等）
   - **開始時間**：選擇請假開始日期與時間
   - **結束時間**：選擇請假結束日期與時間
   - **請假原因**：填寫請假事由
3. 確認假別額度是否足夠
4. 點擊「送出申請」

### 假別類型
| 假別 | 說明 | 額度計算 |
|------|------|----------|
| 特休 | 年度特別休假 | 依年資計算 |
| 病假 | 因病需要休養 | 年度 30 天 |
| 事假 | 因私事需請假 | 年度 14 天 |
| 公假 | 公務出差或訓練 | 不扣額度 |
| 婚假 | 結婚休假 | 8 天 |
| 喪假 | 喪親休假 | 依親等 3-8 天 |
| 補休 | 加班換休 | 依加班時數 |

### 查看假別額度
- 首頁「假別額度」區塊顯示各假別剩餘時數
- 點擊「📊 查看報表」→「假別額度」可查看詳細資訊

### 查看申請狀態
- 首頁「請假記錄」區塊顯示最近申請
- 狀態說明：
  - 🟡 待審批：等待主管審核
  - 🟢 已核准：申請已通過
  - 🔴 已駁回：申請被拒絕

## 加班申請

### 申請步驟
1. 點擊首頁「⏰ 申請加班」按鈕
2. 填寫加班表單：
   - **加班日期**：選擇加班日期
   - **開始時間**：加班開始時間
   - **結束時間**：加班結束時間
   - **加班原因**：說明加班事由
   - **補償方式**：選擇「加班費」或「補休」
3. 點擊「送出申請」

### 補償方式
| 方式 | 說明 |
|------|------|
| 加班費 | 依勞基法規定計算加班費 |
| 補休 | 轉換為補休時數，可用於請假 |

### 加班時數計算
- 平日加班：前 2 小時 1.34 倍，之後 1.67 倍
- 休息日加班：前 2 小時 1.34 倍，之後 1.67 倍
- 例假日加班：2 倍工資

## 補打卡申請

### 適用情況
- 忘記打卡
- 手機故障無法打卡
- 外出公務未能即時打卡

### 申請限制
- 只能申請 **7 天內** 的補打卡
- 年度補打卡額度有限（預設 12 次/年）
- 需填寫詳細原因說明

### 申請步驟
1. 點擊首頁「🔧 補打卡」按鈕
2. 查看剩餘額度
3. 填寫補打卡表單：
   - **日期**：選擇需要補打卡的日期
   - **類型**：補上班打卡 / 補下班打卡 / 補全日打卡
   - **時間**：填寫實際的上班/下班時間
   - **原因**：說明未能正常打卡的原因
4. 點擊「送出申請」

### 查看補打卡記錄
首頁下方「補打卡記錄」區塊顯示所有申請紀錄。

## 主管功能

> 僅限部門主管、HR、總經理可使用

### 開啟主管功能
點擊頁首的「主管功能」按鈕

### 功能說明

#### 1. 出勤總覽
- 查看部門員工當日出勤狀態
- 顯示已打卡/未打卡人數
- 異常出勤提醒

#### 2. 批次審批
審批下屬的各類申請：
- **請假審批**：核准或駁回請假申請
- **加班審批**：核准或駁回加班申請
- **補打卡審批**：核准或駁回補打卡申請

審批操作：
1. 選擇要處理的申請（可多選）
2. 點擊「批准」或「駁回」
3. 填寫審批意見（選填）
4. 確認送出

#### 3. 部門報表
- 部門出勤統計
- 員工請假彙總
- 加班時數統計
- 異常出勤報告

## 常見問題

### Q1: 打卡顯示「位置超出範圍」？
**A:** 請確認：
1. 您確實在公司附近
2. GPS 定位是否準確（可能有誤差）
3. 若持續發生，請聯繫 HR 確認公司 GPS 範圍設定

### Q2: 忘記打卡怎麼辦？
**A:** 請使用「補打卡」功能申請，需在 7 天內提出，並說明原因等待主管審批。

### Q3: 假別額度不足怎麼辦？
**A:**
1. 確認是否有其他假別可使用
2. 若有加班可選擇「補休」方式累積額度
3. 聯繫 HR 確認額度是否正確

### Q4: 申請被駁回後可以重新申請嗎？
**A:** 可以。修正申請內容後重新提交即可。

### Q5: 為什麼看不到「主管功能」按鈕？
**A:** 只有被設定為主管角色的員工才能看到此功能。若您是主管但看不到，請聯繫 HR 確認角色設定。

### Q6: iOS 裝置無法使用相機掃描？
**A:** iOS 裝置需注意：
1. 必須使用 Safari 瀏覽器
2. 必須透過 HTTPS 連線
3. 需允許相機權限
4. 若仍無法使用，可改用「圖片上傳」方式
